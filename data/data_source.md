# Data Source

## Raw Database
The raw FPA FOD SQLite database (777MB) is not included in this 
repository due to GitHub file size limits.

**Download from Kaggle:**
https://www.kaggle.com/datasets/rtatman/188-million-us-wildfires

**File name:** FPA_FOD_20170508.sqlite  
**Place in:** this `data/` folder

## Cleaned Dataset
`fires_sample_10k.csv` — a 10,000 row random sample of the cleaned 
analysis dataset showing the data structure and column definitions.

The full cleaned dataset (1,880,465 rows) is generated by running 
`scripts/01_data_extract.R` after downloading the raw database.

## Column Definitions
| Column | Description |
|--------|-------------|
| FIRE_YEAR | Year fire was discovered |
| DISCOVERY_DOY | Day of year fire was discovered |
| STAT_CAUSE_DESCR | Statistical cause of fire |
| FIRE_SIZE | Size of fire in acres |
| FIRE_SIZE_CLASS | Size classification A (smallest) to G (largest) |
| STATE | U.S. state abbreviation |
| LATITUDE | Fire location latitude |
| LONGITUDE | Fire location longitude |
| OWNER_DESCR | Land ownership description |
| MONTH | Month of discovery (derived) |
| MONTH_NAME | Month name abbreviation (derived) |
| CAUSE_TYPE | Human or Natural (derived) |
| SIZE_CLASS_LABEL | Readable size class label (derived) |
